Cool Reads and Resources

Below are some resources I have used in the past few years. I plan to update this list with new textbooks after each semester of using that book, and will continuously update the list with other resources I use for projects and self-learning.

3D Graphics from Scratch

A blog I referenced when working on 3D graphics from scratch in javascript (see the cool polyhedrons in the background? Yea thats being computed in real-time on your device right now, sorry for the CPU overhead).

Visit
SIMD Programming

A practical intro to SIMD parallel computing.

Read
Mathematics for Computer Science

This was the textbook used in my discrete math class, and I find myself coming back to it many times for reference and to brush up on topics. I highly recommend this book for reference and self-learning.

Read
Improving Trust in Software through Diverse Double-Compiling and Reproducible Builds

A thesis on a method to counter trusting-trust attacks.

Read
Control

A chapter from Cornell's CS 4750 course. An introduction to control, I read this to get an idea of how pure pursuit controllers work.

Read
Reflections on Trusting Trust

An interesting read on trusting compilers, and the impact a breech on one could have.

Read
Computer Systems: A Programmer's Perspective

This textbook was used for my computer architecture and systems programming courses, and it is great at explaining computing systems from the ground up, while maintaining simplicity. I highly recommend this to anyone looking for a solid introduction to low-level computing topics.

Read
Neural Networks and Deep Learning

An amazing online book that explains the concept of neural networks and dives into deep learning topics.

Visit
Learn about USB

A descriptive website on the workings of the USB 2.0 protocol.

Kalman Filter

A well put together website explaining the motivation for and implementation of a kalman filter.

Visit
SQL Injection and More

A blog containing direct examples and methods for SQL injection attacks, and some Google Dorks. The site also contains various other hacking topics.
